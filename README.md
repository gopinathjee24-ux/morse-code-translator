# Morse Code Translator Using Python

##  Project Overview
This project converts English text to Morse code and Morse code back to English. The translator uses a dictionary mapping and handles alphabets, digits, and spaces.

##  Features
✔ Text → Morse code conversion  
✔ Morse code → Text conversion  
✔ Simple user menu  
✔ Works offline  

##  How to Run
```bash
python morse.py
